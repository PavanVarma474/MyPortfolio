Change Detection in Satellite Images using Deep Learning
This Project explores a GAN based method to find and mark key changes in satellite imagery. The proposed architecture involves a generator that creates plausible modification and a discriminator that finetunes these by discriminating the real from the artificial. Then, thresholding to the images results in the creation of a binary mask, which visually shows the detected changes. The model’s accuracy, precision, recall, and F1 measure against ground-truth data are demonstrated, thus illustrating the potential of GANs to reliably automate change detection in satellite images.
